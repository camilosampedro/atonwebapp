Introduction

This project aims to be a seed for single-page user driven web apps.

  • Persistence: Slick
  • Presentation: AngluarJS and Bootstrap
  • REST: Play Framework
  • Security: SecureSocial

Setup

  1. Download the project from https://github.com/spass/spass
  2. Navigate to the folder
  3. Execute ./activator run
  4. Enjoy
